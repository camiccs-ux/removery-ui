# Typography

Removery UI uses two font families for visual hierarchy, both confirmed directly from the Removery Library Figma file via `get_design_context` — unlike colors (still pending sync), these values were read from the real Figma source, not estimated.

## Font Families

<SimpleTable
  mono

`font-mono-text`, `font-serif-text`, and `font-caveat-text` also exist as variables in the Removery Library, but aren't applied anywhere in R7 or the public funnel today — not included above since they're unused.

## Scale pending confirmation

The library has 6 more `display` steps (`sm`, `md`, `lg`, `xl`, `2xl`, `3xl`) and the full `body` scale (`text-xs`, `text-sm`, `text-lg`, `text-xl`, each in `normal`/`semibold`/`bold`) as named composite styles — same idea as `body/text-base/semibold` above, just not pulled yet. Confirming each one requires opening one text layer per style in Figma (no batch lookup available).

If you want to close out the full scale, send the link to another text layer (Copy link to selection, on a layer styled `display/lg` or `body/text-xs/normal`, for example) and this page will keep growing.
