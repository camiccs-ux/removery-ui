# Dark Mode

O sistema usa variáveis CSS por classe (`.dark`), o mesmo padrão do shadcn/ui — cada token tem um valor em `:root` (light) e outro em `.dark` (dark), definidos em `src/index.css`.

## Como ativar

```tsx
// No elemento raiz do app (ou de uma seção específica):
<html className={isDarkMode ? "dark" : ""}>
```

Todos os componentes já usam os tokens semânticos (`bg-background`, `text-foreground`, `bg-primary`, etc.) em vez de cores fixas — então ativar `.dark` na raiz já propaga para todo o app automaticamente, sem precisar tocar em nenhum componente individual.

## Testar no Storybook

Use o seletor de tema (ícone de sol/lua) na barra de ferramentas no topo do Storybook — ele aplica a classe `.dark` no preview em tempo real.

## Nota sobre a collection `🖼️ Theme`

A Removery Library tem uma segunda collection de variáveis (`🖼️ Theme`), reservada para uma possível marca/tema futuro — **não é o dark mode** e não deve ser confundida com ele. A collection que este projeto reflete é a `🌗 Theme Mode`, que é a publicada e em uso hoje.
